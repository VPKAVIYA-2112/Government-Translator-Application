Standardized government document translation web system for indian languages

Government Translator Application

A web-based AI-powered document translation system that helps users translate government-related documents into multiple Indian languages. The application supports PDF and Word document processing, voice interaction, and real-time translation through a modern and responsive interface.

🚀 **Features**
📄 Upload and process PDF/DOCX documents
🌐 Translate content into multiple Indian languages
🎤 Speech-to-text voice input
🔊 Text-to-speech translated output
⚡ Real-time document processing
🎨 Responsive and user-friendly interface
🧠 AI-assisted translation workflow
